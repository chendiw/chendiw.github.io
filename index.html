Finite Difference Operator

In this part, I first calculated the partial derivatives of the image "camaraman" with respect to vectors [[1, -1]] and [[1], [-1]]. Then, gradient magnitude is discovered by calculating the square root of sum of the two partial derivative squared. The result is then binarized using a threshold value of 0.1.

Derivative of Gaussian Filter

Since the direct derivative of the original image captures a lot of noise, a gaussian filter is applied to the original image to remove noises before the partial derivatives are taken and the magnitude is calculated as in the previous part. The gaussian kernel is created with parameters: size = 5, sigma = 1, after a couple of experiments.

I also experimented with taking the derivatives of the gaussian filter first and then performing one convolution. The resulting edge image (last image in the second row) is similar to that created using the first approach.

Image Straightening

In this part, automatic image straightening is implemented. A range of rotation angles (i.e. -30 to 30 degrees) are experimented with, and for each angle, I record the number of horizontal/vertical edges in the center of the rotated image. The number of horizontal/vertical edges is counted from the number of 0/90/180 degree gradient angles. The rotation with the maximum number of horizontal/vertical edges is picked and considered the rotation angle needed to straighten the image.

Image Sharpening

In this part, automatic image sharpening is implemented. To sharpen the image, I utilized an unsharp mask filter which is built on top of a gaussian filter that creates a low pass filter that blurs the image. Then, I subtract the blurred image from the original image to get a high frequency image, multiply by some pre-set weights and then add back the original image to achieve sharpening.

I also experimented with first blurring an already-sharp image and then resharpening it. However, this process doesn't give us back the original image. One possible reason is that a lot of pixel-level details are lost during blurring, which are never added back in the sharpening phase. Sharpening doesn't create extra information about the pixels!

Hybrid Images

In this part, I created hybrid images taking advantage of how human perception works. Humans are naturally more sensitive to high frequencies if looking at an image from a short distance, while more sensitive to low frequencies when lookinf from afar. Thus, this set of hybrid images are created by aligning a low-frequency image with another high-frequency image, creating an illusion of the image being one thing at a short distance but another at a longer distance.

Gaussian & Laplacian Stacks

In this part, I implemented Gaussian and Laplacian stacks. Gaussian stacks are achieved from iteratively applying the gaussian filter to the previous image. Laplacian stacks are achieved as a difference between two neighboring gaussian images.
